# Scratch Tutorial

### 1. Create virtual environment
    python -m venv .venv

### 2. Activate virtual environment
- On MACOS:

    source .venv/bin/activate

- On Windows:

    source .venv/Scripts/activate

### Pull source code
    git clone https://github.com/phich82/mai_scratch.git

### Write new codes
    git status
    
    git add .

    git commit -m "add more codes"

    git push
